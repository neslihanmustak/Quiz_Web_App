# QuizWebApp

## Local Config

This project uses `Web.config` for configuration. Do not commit real connection strings.

Steps:
1. Use `MvcQuizSonDeneme/MvcQuizSonDeneme/Web.config.example` as the template.
2. Edit `MvcQuizSonDeneme/MvcQuizSonDeneme/Web.config` locally with your real database values.

`Web.config` in the repo contains only safe defaults. Your local edits should stay uncommitted.
